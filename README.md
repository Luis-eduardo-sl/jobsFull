<h1 align="center">Para rodar o projeto</h1>
<br>

- FrontEnd:<br>
```bash
 cd frontend
```
```bash
 npm install
```
```bash
 npm run tunnel
```
<br>
O BackEnd esta online no render, mas caso decida rodar localmente pode mudar as urls para http://localhost:3333 e o resto continua igual
<br>
<br>

- Backend:<br> 
```bash
 cd backend
```
```bash
 npm install
```
```bash
 npx prisma db push
```
```bash
 npx prisma generate
```
```bash
 npm run dev
```
